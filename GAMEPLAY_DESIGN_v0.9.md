# 全零〈オムニル〉v0.9 ゲーム性設計

## 戦闘の基準

- 基本は3人固定パーティのコマンド戦闘。
- 通常戦は、装備・習熟・料理が現在地方に見合っている時に**おおむね4〜5行動**で決着する想定。
- 上級探索、レア敵、ボスは装備効果・バフ・デバフ・道具の自動設定を使うことを前提にする。
- 道具はターンを消費しない。道具を使った後も同じキャラクターが技か通常攻撃を選べる。

## 成長の役割

| 要素 | 役割 |
|---|---|
| キャラクターLv | 基礎能力、パネルPT、技の土台 |
| 冒険者Lv | 地方、施設品、料理、依頼、スタミナ上限の解放 |
| 技習熟 | 技単体の威力、回復、持続、極意効果 |
| パネル | ビルドの方向性と新技・パッシブ |
| 装備 | 戦い方を変える特殊効果 |
| 食堂 | 地方周回に合わせた時間制限バフ |

## 装備の基本ルール

装備枠は頭、体、腕、足、右手、左手、装飾1、装飾2。

- 片手武器は右手・左手へ装備可能。
- 片手盾は左手へ装備。
- 両手武器、両手盾は右手から装備し、左手を必ず占有する。
- 同じ装備は全員で共有し、一人にしか装備できない。

## オート道具

オート道具設定は、手動AUTOとオート探索の両方で使う共通ルール。

1. 蘇生
2. HP回復
3. MP回復
4. 結界
5. 技・通常攻撃

の優先順で判断する。各カテゴリに一戦／一周の上限を設定し、消費し過ぎを防ぐ。

## 経済

- 装備は高価格帯。地方の探索・依頼・素材売却で資金を貯める。
- 消耗品は個数指定でまとめ買い・まとめ製造できる。
- 宿屋料金は冒険者Lvに連動。長期進行ほど滞在費も上がる。
- 繰り返し依頼は素材・G・周回導線を主目的とし、冒険者Lvを急激に上げない。
