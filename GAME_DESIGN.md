# 全零〈オムニル〉— 星なき始まり — 実装設計 v0.9

## 実装済みの柱

- 十地方・地方地図・町・探索地
- 地方ごとのギルド、ショップ、製造所、食堂、宿屋、買取所
- 3人固定のコマンド戦闘とAUTO
- 冒険者Lv.1〜100、キャラクターLv、パネル、技習熟
- キークエストとテリオス解放導線
- 8部位装備と両手装備の排他
- スタミナ、オート探索、共通オート道具設定

## v0.9で固定した方針

1. 反復依頼は周回の補助であり、ストーリー・地方解放を飛ばす経験源にはしない。
2. 後半はレベルだけでは突破せず、装備の複合効果、料理、パネル、技習熟を組み合わせる。
3. 道具は使い惜しみを減らすため、戦闘ターンを消費しない。
4. オートは便利さを優先しつつ、使用数上限で資源管理を残す。
5. 静的Webゲームとして、外部ライブラリ・外部音源なしでGitHub Pages上から動作する。
