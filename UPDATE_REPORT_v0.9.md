# 全零〈オムニル〉 v0.9 更新報告

## 今回の主題

v0.9は、長時間周回しても成長が早過ぎず、装備・食事・パネル・道具設定に意味が残るよう、**戦闘・経済・自動化をまとめて再設計**した更新です。

| 区分 | 変更内容 |
|---|---|
| 装備 | 両手装備の排他を完全化。特殊効果・価格・能力値を再調整 |
| 経済 | 宿屋料金をLv連動化。装備価格を大幅上昇 |
| 成長 | 繰り返し依頼の冒険者経験を抑制、敵経験とLv曲線を長期向けに調整 |
| 戦闘 | 道具はターン消費なし。通常戦は4〜5行動程度を目安に再配分 |
| 自動化 | オート戦闘／オート探索の道具使用を共通設定へ集約 |
| 音 | 攻撃・魔法・被弾・回復・強化・弱体の効果音を追加 |

## 1. 両手装備の修正

- 両手武器・両手盾を右手へ装備した時、左手スロットを必ず空欄にします。
- 両手装備中は、左手へ新しい装備を付けられません。
- 左手欄には「右手の両手装備で使用中」と明示します。
- v0.8以前のセーブで両手装備と左手装備が同居していた場合も、ロード時に左手を自動解除します。

## 2. 長期成長・難易度

- 冒険者Lv.1〜100の必要冒険者経験を、中後半ほど大きく伸びる曲線へ変更。
- 繰り返し依頼の冒険者経験は旧値の約28%へ減少。
- 敵のHP・攻撃・魔力・防御を上げ、敵撃破経験は下げました。
- ボスは通常敵より耐久・火力を強くし、報酬経験は通常敵より高めに維持しています。
- 高地方は、装備の特殊効果、料理、パネル、技習熟、道具運用を整える前提です。

## 3. 装備・経済

- 全205装備の基本能力値を強化。
- 装備価格を大幅に引き上げ、地方で稼ぐ理由を作りました。
- 装備には以下のような複合効果を追加・拡張しています。
  - 与ダメージ、物理／魔法威力、会心率、会心威力
  - ボス特効、弱体敵特効、瀕死時火力、防御貫通
  - 被ダメージ軽減、状態異常耐性、障壁開始、かばう軽減
  - 回復技、道具回復、MP消費、戦闘後回復
  - 経験値、G、希少発見、習熟度、探索スタミナ軽減
- 中盤以降の装備は、能力値に加えて二種類以上の特殊効果を持つようにしています。
- 宿屋料金は冒険者Lvに応じて上昇します。

## 4. 戦闘道具とオート設定

戦闘中の道具は、行動ターンを消費しません。HP回復、MP回復、蘇生、結界を状況に応じて使い、そこから技や攻撃を選択できます。

メニュー → **オート道具設定** で、次を個別に設定できます。

- HP回復を始めるHP割合、使用する道具、一戦の最大使用数
- MP回復を始めるMP割合、使用する道具、一戦の最大使用数
- 蘇生の有無、使用する道具、一戦の最大使用数
- 結界を張るHP割合、使用する道具、一戦の最大使用数
- オート探索中の戦闘で上記設定を使うか
- スタミナ回復薬の種類と、一周あたりの最大使用数

旧「オート探索専用の活力剤設定」は廃止し、すべてこの共通設定へ統合しました。

## 5. 個数指定

ショップで購入する消耗品と、製造する消耗品は個数を入力してまとめて処理できます。

- ショップ：1〜99個
- 製造：1〜99回
- 装備は共有かつ一品物のため、従来どおり1個ずつ購入・製造です。

## 6. 効果音

外部音源を使わない生成音のまま、下記を追加しました。

- 通常攻撃
- 魔法・特殊攻撃
- 被弾
- 回復
- 強化
- 弱体
- 障壁
- 回避
- 戦闘開始
