# v0.7 テスト手順

1. リンドホルムのギルドを開き、`受注可能をすべて受注` を押す
2. 素材・討伐条件を満たしたら、`報告` タブで `報告可能をまとめて報告` を押す
3. `繰返・納品` と `繰返・討伐` を切り替え、再受注の手間を確認する
4. 道具屋で共通装備を購入し、仲間 → 装備変更から各部位へ装備する
5. 活力剤を持った状態でオート探索を開き、下限・自動使用数を設定する
6. 冒険者レベルを上げ、白霜北域 → 緋火海岸 → 暮影峡谷の順に地方を確認する
7. 各地方のショップ、製造所、ギルドで品揃え・依頼が異なることを確認する

フィードバックでは、依頼の手間、装備の選びやすさ、地方の報酬差、スタミナの回り方、敵の硬さを優先して見てください。
