# 全零〈オムニル〉 v0.7 まとめて更新報告

## 1. ギルドの依頼報告を一括化

各地方の冒険者ギルドでは、依頼ページを閉じずに受注・報告を行えます。

- `受注可能をすべて受注`：その地方で受けられる一回限り・繰り返し依頼をまとめて受注
- `報告可能をまとめて報告`：条件を満たした納品・討伐依頼を一度に報告
- 繰り返し依頼を `繰返・納品` と `繰返・討伐` に分離
- 報告後も同じタブを維持し、繰り返し依頼はすぐに再受注可能

## 2. 地方ごとに依頼を分離

ギルド依頼は地域で分離しました。

| 地方 | 拠点 | 主な依頼テーマ |
|---|---|---|
| リンドホルム地方 | 宿場町リンドホルム | 草原・森・遺構の基礎依頼 |
| 白霜北域 | 霜門の集落 | 雪原、観測所、凍った記録 |
| 緋火海岸 | 熾港アグニラ | 火山殻、潮影、火口、沈み書庫 |
| 暮影峡谷 | 暮影の渡り場 | 虚骨、剣墓、二振りの剣、調律の祭壇 |

一回限り依頼と、地方ごとに異なる繰り返し依頼を実装しています。

## 3. 装備を6部位へ拡張

仲間メニューから、全員分の装備を変更できます。

1. 頭
2. 体
3. 腕
4. 足
5. 装飾1
6. 装飾2

装備品は原則として全員が使えます。虹全・白零・黒零の専用装備のみ、対象キャラクターだけが装備できます。

装備の特殊効果例：

- 戦闘開始時に障壁
- 技MP消費を軽減
- 回復効果上昇
- 弱体中の敵への与ダメージ上昇
- 戦闘勝利後のHP回復
- 希少素材・隠し発見率上昇
- 所持金・経験値獲得量上昇
- 被ダメージ軽減

ショップ購入装備と地方製造装備は、完成後に自動装備されません。仲間画面の `装備変更` から付け替えます。

## 5. スタミナの仕様変更

- 初期スタミナ：100
- 冒険者レベル上昇で上限+50
- 探索・戦闘で消費
- 自然回復、宿屋、活力剤で回復
- 活力剤は上限を超えて回復可能
- スタミナ0では探索／オート探索を開始できない
- **スタミナ0による撤退・敗北ペナルティは廃止**

オート探索では、下限・活力剤の種類・自動使用回数を設定できます。次の探索または戦闘で下限を下回りそうな時だけ自動使用し、指定回数を使い切るまで探索を継続します。

## 6. 既存セーブの扱い

v0.6以前のセーブは、所持金・素材・レベル・依頼進行をできる限り維持します。

旧1枠装備は、該当するv0.7装備へ移行し、6部位のいずれかに配置されます。装備移行後は、仲間メニューから好みに合わせて調整してください。
