# 全零〈オムニル〉— 星なき始まり — v0.7

ドット絵風の画面、3人固定パーティ、コマンドバトル、探索・依頼・製造・装備・成長パネルを備えた、ブラウザで動くRPG試作版です。

## 起動

`index.html` をブラウザで開くか、GitHub Pages 上の公開URLへアクセスします。外部ライブラリ・外部画像・外部音源は不要です。

## v0.7 の主な追加

- 地方ギルドごとの依頼板と、受注・報告の一括処理
- 納品系／討伐系を分けた繰り返し依頼
- 6部位装備（頭／体／腕／足／装飾1／装飾2）と仲間画面での付け替え
- 共通装備と虹全・白零・黒零の専用装備
- 緋火海岸／暮影峡谷を追加し、実装地域を4地方へ拡張
- 地方別のショップ、製造、敵、レア敵、ボス、素材、隠し発見
- スタミナ0は探索を始められないだけで、撤退・敗北ペナルティなし

## セーブ

進行はブラウザのローカルストレージへ自動保存されます。v0.6以前のセーブを読み込む場合、装備はv0.7の6部位制へ移行します。

## 主な操作

- **世界**：世界地図と地方地図を切り替え、探索地・町へ移動
- **仲間**：ステータス、成長パネル、装備変更
- **依頼**：全地方の依頼一覧。各地方の町では地方専用のギルド依頼を扱う
- **荷袋**：道具・素材・所有装備の確認

詳細は `UPDATE_REPORT_v0.7.md` と `GAMEPLAY_DESIGN_v0.7.md` を参照してください。
