# 全零〈オムニル〉— 星なき始まり — weekday test v0.6

白零・黒零・虹全の三人で進む、ドット絵コマンドRPGの平日テスト版です。

## 起動

`index.html` をブラウザで開くか、GitHub Pages 上の公開URLへアクセスしてください。

> セーブデータはブラウザの `localStorage` に保存されます。v0.5までのデータは初回起動時に移行され、所持金・素材・冒険者経験などを引き継ぎます。

## v0.6 の主な追加内容

- 世界地図＋地方地図の二段階構成
- リンドホルム地方と白霜北域、計13地点
- 一回限り27件／繰り返し23件、計50件の依頼
- 初級・中級・上級の探索難易度
- オート探索時の活力剤自動使用設定
- 道具33種、製造20種、固有装備12種
- 敵21種、レア敵、ボス、メタル系、ゴールド系、隠し発見
- 技習熟Lv.1〜10、既存の成長パネル・テリオス盤
- 地形ごとに切り替わるゲーム内生成BGMと効果音

## 今回のテストで見てほしいこと

1. オート探索が「下限スタミナ＋活力剤使用数」の意図どおりに動くか
2. 難易度ごとの消費・報酬・敵の強さが納得感あるか
3. 依頼、ショップ、製造、装備の回転が楽しいか
4. 敵やレア敵、ボスの出現頻度が適切か
5. 技、パネル、習熟度の強さと育成速度が気持ちいいか

具体的な記入項目は `WEEKDAY_FEEDBACK.md` にまとめています。
