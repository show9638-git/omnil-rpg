# 全零〈オムニル〉— 星なき始まり — prototype v0.4

白零・黒零・虹全の三人で始める、ドット絵コマンドRPGのプレイアブル試作版です。

## 起動

`index.html` をブラウザで開くだけで起動します。外部ライブラリ、外部画像、ビルド手順は不要です。

スマホで公開する場合は、フォルダ一式を GitHub Pages などの静的ホスティングへ配置してください。

## v0.4の主な追加・変更

- 成長パネルを、全ノード重なり表示から**系統タブ＋選択式の星盤**へ全面変更
  - パネルはすべてタップして詳細を確認可能
  - 選択後、効果・解放条件・必要PTを確認してから解放可能
  - スマホでは各系統を2列表示し、文字が重ならない構成
- 「冒険者ランク」を **冒険者レベル** に名称統一
- 冒険者レベルに応じてスタミナ上限が増加
  - Lv.1: 60 / Lv.2: 70 / Lv.3: 80 / Lv.4: 90 / Lv.5: 100
- スタミナを追加
  - 探索・戦闘開始時に消費
  - 3分ごとに1回復
  - 宿屋で全回復
  - 活力の小瓶で25回復
  - 0になると撤退扱い（15G喪失、HP・MP25%で帰還）
- オート探索を、回数固定から**スタミナ下限指定型**へ変更
  - 指定した下限を守りながら自動で継続
  - 探索・戦闘のスタミナ消費は2倍
  - 下限0を指定した場合でも、力尽き事故を避けるため残り1で安全停止
- 依頼を15件に拡張
  - 一回限りの依頼 8件
  - 繰り返し受注できる依頼 7件
- BGM用の `js/audio.js` を読み込み、場面ごとのゲーム内生成音を有効化

## 実装済みの遊び

- ドット絵の広域マップと、地域別ロケーション画面
- 宿場町リンドホルム：ギルド、道具屋、製造所、宿屋、素材買取所
- 風渡る草原、囁きの森、落星の遺構
- ランダム探索：素材、所持金、敵遭遇
- 3人固定パーティのコマンドバトル／AUTO戦闘
- 技ごとの習熟度（Lv.1～10）
- 3人それぞれの81パネル
  - 技30／パッシブ10／能力値40／根源1
- 種族特性 各3つ
- 素材収集、売却、製造、回復アイテム
- LocalStorage 自動セーブ／手動セーブ／リセット

## ファイル構成

```text
index.html          起動ページ
style.css           見た目とレスポンシブUI
js/data.js          敵・依頼・パネル・アイテムなどのゲームデータ
js/audio.js         Web AudioによるBGM・効果音
js/game.js          画面遷移、探索、戦闘、スタミナ、セーブ
```
