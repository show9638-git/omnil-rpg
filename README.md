# 全零〈オムニル〉— 星なき始まり — v0.9

白零・黒零・虹全の三人で進む、ドット絵コマンドRPGのプレイアブル試作版です。

## 起動

`index.html` をブラウザで開くか、GitHub Pages に配置して遊びます。外部ライブラリ・外部画像・外部音源は使用していません。

## v0.9 の主な内容

- 冒険者Lv.100、十地方、地方別ショップ・製造・食堂・依頼・キークエスト
- 両手装備の左右手排他を完全化
- 装備205種の能力・特殊効果・価格を再調整
- 宿屋料金を冒険者Lv連動化
- 繰り返し依頼の経験を抑え、敵の強さ・経験・成長曲線を長期向けに再調整
- 戦闘中の道具使用はターンを消費しない
- HP／MP／蘇生／結界／活力剤を使うオート道具設定
- ショップ・製造で消耗品の個数指定
- 攻撃、魔法、被弾、回復、強化、弱体などの生成効果音

## セーブ

ブラウザのLocalStorageに自動保存します。v0.8以前のセーブは可能な範囲で引き継ぎ、初回にv0.9形式へ移行します。
