# 全零〈オムニル〉— 星なき始まり — prototype v0.3

白零・黒零・虹全の三人で始める、ドット絵コマンドRPGのプレイアブル試作版です。

## 起動

`index.html` をブラウザで開くだけで起動します。外部ライブラリ、ビルド手順、外部アセットは不要です。

## v0.3の追加内容

- オート戦闘（戦闘中に随時オン／オフ）
- オート探索×5（戦闘遭遇時は自動戦闘へ移行）
- 技ごとの習熟度（最大Lv.10）
- 虹全／白零／黒零それぞれに、技30・パッシブ10・能力値40の大型成長パネル
- テリオス覚醒後に入る追加パネルの下地
- 障壁、再生、かばう、防御低下、攻撃低下、鈍化、沈黙、拘束などの拡張戦闘効果
- 3人それぞれ3つの種族特性

## 主なファイル

- `index.html`：起動ページ
- `style.css`：画面・パネル盤・戦闘UI
- `js/data.js`：キャラ、90技、30パッシブ、120能力値パネル、敵、依頼、素材
- `js/game.js`：戦闘、オート、探索、成長、セーブ
- `js/audio.js`：外部音源なしのBGM・効果音
- `GAMEPLAY_DESIGN_v0.3.md`：今回のゲーム性・パネル・性能一覧
