# 全零〈オムニル〉— 星なき始まり — prototype v0.5

白零・黒零・虹全の三人で旅を始める、ブラウザ完結型のドット絵コマンドRPG試作版です。

## 起動

`index.html` をブラウザで開くか、GitHub Pagesで公開してください。

外部ライブラリ・外部画像・外部音源は不要です。セーブデータはブラウザのローカルストレージに保存されます。

## v0.5の主な内容

- リンドホルム**地方地図**から町・探索地を選択
- 初級／中級／上級探索
- 3人固定パーティのコマンド戦闘とAUTO戦闘
- 下限指定で継続するオート探索
- スタミナ、自然回復、宿屋回復、上限超過可能な回復薬
- 冒険者レベルによる行動範囲・ST上限の拡張
- 一回限り13件＋繰り返し11件の依頼
- 依頼一括受注、受注中／達成済みフィルタ
- 上から順に取得する成長パネル
- 技の習熟度Lv.1〜10

## GitHubへ更新する場合

1. 配布ZIPを展開する。
2. `index.html`、`style.css`、`js` フォルダ、各 `.md` をリポジトリ直下に上書きアップロードする。
3. GitHub Pagesの反映後、ブラウザを再読み込みする。

旧版のセーブデータは可能な範囲で移行します。v0.4以前のセーブを読み込んだ場合、スタミナは最低100へ補正され、各キャラに成長PT+3が追加されます。

## 現在は仮素材

マップ、人物、敵はゲーム性検証用の仮ドットです。ゲームループと育成が固まった後、地域ごとの地形タイル、人物アニメーション、敵アニメーション、正式UI、音楽演出を本制作します。
