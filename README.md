# 全零〈オムニル〉— 星なき始まり — prototype v0.1

白零・黒零・虹全の三人で始める、ドット絵コマンドRPGのプレイアブル試作版です。

## 起動

`index.html` をブラウザで開くだけで起動します。外部ライブラリ・外部画像・ビルド手順は不要です。

スマホで公開する場合は、フォルダ一式を GitHub Pages などの静的ホスティングへ配置してください。

## 実装済み

- タイトル／オープニング
- ドット絵の広域マップと、地域別ロケーション画面
- 宿場町リンドホルム：ギルド、道具屋、製造所、宿屋、素材買取所
- 風渡る草原、囁きの森、落星の遺構（冒険者ランク E で解放）
- ランダム探索：素材、所持金、敵遭遇
- 3人固定パーティのコマンドバトル
- 5種類の敵と、苔牙の獣王ボス
- 経験値・レベルアップ・パネルポイント
- キャラ固有の成長パネル、種族特性、固有技
- ギルド依頼4件、冒険者ランク、地域解放
- 素材収集、売却、製造、回復アイテム
- LocalStorage 自動セーブ／手動セーブ／リセット

## 制作上の前提

- 人物イラストは後から差し替えられる「ステータス用肖像枠」として実装しています。現状は外部素材不要の仮ピクセル肖像です。
- すべてのゲームデータは `js/data.js` にまとめており、地域・敵・依頼・パネル・アイテムを増やしやすい構成です。
- 見た目と挙動の本体は `style.css` と `js/game.js` です。更新時はファイル全量を置き換える運用に対応しています。
