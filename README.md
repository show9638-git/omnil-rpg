# 全零〈オムニル〉— 星なき始まり — v0.8

白零・黒零・虹全の三人で進む、ドット絵コマンドRPGのプレイアブル試作版です。

## 起動

`index.html` をブラウザで開くか、GitHub Pages に配置して遊びます。外部ライブラリ・外部画像・外部音源は使用していません。

## v0.8 の主な内容

- 冒険者レベル上限を **Lv.100** に拡張
- 世界地図から選ぶ **10地方** と、地方ごとのマップ・町・探索地を追加
- キークエストで各章を進め、後半に **テリオス覚醒** へ到達する構成
- 各地方の食堂に10種、計100種の料理を追加
- 右手・左手を含む **8装備枠**。片手／両手武器・盾の挙動に対応
- 装備205種、製造114件、依頼148件、敵61種、技99種
- 未使用技にも戦闘終了時に少量の習熟経験を付与
- パネル系統を選んで一括解放できる操作を追加

## セーブ

ブラウザの LocalStorage に自動保存します。v0.7以前のセーブは可能な範囲で引き継ぎ、初回にv0.8形式へ移行します。
