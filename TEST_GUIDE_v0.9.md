# v0.9 テスト手順

## まず確認してほしい導線

1. 仲間 → 任意のキャラ → 装備を変更する。
2. 右手に両手武器または両手盾を装備する。
3. 左手欄が「右手の両手装備で使用中」になり、装備候補を押しても変更できないことを確認する。
4. 右手から両手装備を外し、左手装備が再び選べることを確認する。

## オート道具

1. メニュー → オート道具設定を開く。
2. HP回復薬・MP回復薬・蘇生薬・結界珠と、使用上限を設定する。
3. 探索地でAUTOを開始し、HPが閾値を下回った時に道具が先に使われることを確認する。
4. オート探索の設定から、スタミナ下限を指定する。
5. スタミナ回復薬と最大使用数を設定した場合、下限を割る前に自動使用することを確認する。

## 価格と成長

1. 宿屋の料金を現在の冒険者Lvで確認する。
2. ショップで回復薬の数量を変え、まとめて購入する。
3. 製造所で消耗品の製造数を変え、素材消費と完成数が対応することを確認する。
4. 繰り返し依頼と一回限り依頼を比較し、繰り返し依頼の冒険者経験が低めであることを確認する。

## フィードバックで見たい点

- 通常戦が短過ぎず、長過ぎないか
- 道具を使った時の安心感と、所持金・補給の重さが噛み合っているか
- オート道具の初期設定が使いやすいか
- 両手装備の説明とUIが直感的か
- 高地方の敵が「準備すれば勝てる」難しさになっているか
