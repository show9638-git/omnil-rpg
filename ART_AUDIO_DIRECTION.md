# 全零〈オムニル〉：音・会話・ドット絵の制作方針

## 1. 会話の基準

### 虹全〈コウゼン〉
- 若い男性として自然に話す。軍人・AI・司令官のような言葉は使わない。
- 二人を指揮するより、気遣いながら「三人で決める」側に立つ。
- 短い台詞でも、会話の受け答えを必ず入れる。
- 例：「大丈夫か」「無理はするな」「俺もそう思う」「行こうか」

### 白零〈ハクレイ〉
- 静かで丁寧。硬すぎず、説明口調にしない。
- 感情を知らないため、最初は自分の内側を確かめるように話す。
- 例：「……わかった」「私が前に出る」「役に立てた？」「よかった」

### 黒零〈コクレイ〉
- 率直で短い。冷たいのではなく、言葉を選ぶ経験が少ない。
- 仲間を守る時だけ、言葉が少し強くなる。
- 例：「邪魔なら、退かせる」「それでいい」「……静かになった」「壊させない」

## 2. 音響の方針

v0.2では外部素材に依存しない Web Audio のチップ・アンビエント音を実装する。
ブラウザの最初の操作後、場面に応じてBGMが切り替わる。

| 場面 | 役割 | 音の印象 |
|---|---|---|
| タイトル | 神話と孤独 | 星、余白、ゆっくりした和声 |
| 世界地図 | 旅の始まり | 前向きだが未知が残る |
| リンドホルム | 安堵と生活 | 木管・鈴のような軽さ |
| 草原 | 風と開放感 | 明るい上行フレーズ |
| 囁きの森 | 迷いと生命 | 低い残響、控えめな旋律 |
| 落星の遺構 | 神話の断片 | 不穏、空間、古い響き |
| 戦闘 | 緊張と決断 | 明確なリズム、短い反復 |
| 勝利 | 一息 | 小さく前を向く終止 |

正式版では、同じ楽曲コンセプトをもとに、オリジナルの OGG/WAV 楽曲と環境音へ置き換える。
音源の採用条件は「自作、または商用利用・改変・ゲーム配布を含む利用条件が明確なもの」に限定する。

## 3. ドット絵の本制作

### 基本規格
- マップタイル：32×32px 基準
- フィールド用キャラ：48×64px 前後
- 戦闘敵：小型 64×64px、中型 96×96px、ボス 128×128px 以上
- UIはドットを崩さない整数倍率で拡大する
- 輪郭・影・光源のルールを先に決め、地域ごとに描き直す

### 色のルール
- 白・黒・虹は三人と神性の象徴。一般の風景には使いすぎない。
- 草原は黄緑・空色・土色を中心に、三人の色が画面で埋もれないようにする。
- 森は深緑と青緑、遺構は青灰と古金を基調にする。
- 虹は常時発光ではなく、「力が揺れた瞬間」にだけ現れる。

### 制作順
1. 世界地図の地形・海・街・街道を描き直す
2. リンドホルムの施設外観と住人を作る
3. 草原・森・遺構のタイルセットを分ける
4. 三人の歩行・待機・攻撃・被弾を作る
5. 敵の輪郭と最低限の待機／被弾アニメーションを作る
6. 戦闘演出、天候、環境エフェクトを追加する

## 4. 重要な約束

仮ドットは「完成品の見た目」ではない。
先に探索・依頼・成長・戦闘の手触りを固め、その後でオムニル固有の絵作りへ差し替える。
最終的には、町ごと・探索地ごとに一目で空気が違うゲームにする。
